-  Vue 
-  Vuex

- ✨Magic ✨


## Installation

todolist requires [Node.js](https://nodejs.org/) v14+ to run.

Install the dependencies and devDependencies and start the server.

```sh
git clone https://github.com/juanperdomolol/VueChallenge.git
cd challeng-typescript
npm i
npm serve
```
